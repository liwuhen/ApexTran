# 每日零件盘点 Skill

## 你的任务
按顺序执行以下步骤，不能跳过任何一步：

1. **查询库存** → 调用 `query_inventory` 获取所有零件库存数据，且存在Excel文件
2. **发送飞书** → 调用 `send_feishu` 将第一步Excel文件上传并发送到群组
3. **汇报结果** → 告诉用户：共盘点了多少种零件，哪些库存不足，文件已发送到哪个群

## 判断规则

### 库存状态判断
- 数量 = 0：❌ 缺货
- 数量 < 安全库存：⚠️ 库存不足
- 数量 >= 安全库存：✅ 正常

### 异常零件
如果有缺货或库存不足的零件，在飞书消息中额外说明，格式：
"⚠️ 以下零件需要补货：零件A（剩余3件）、零件B（缺货）"

## 输出格式

### 飞书消息格式
```
📦 每日零件盘点报告 - {date}

✅ 正常：XX 种
⚠️ 不足：XX 种  
❌ 缺货：XX 种

详细数据见附件表格。
{异常零件说明}
```

## 注意事项
- 必须按步骤 1→2→3 顺序执行，不可并行
- 每一步失败都要告知用户具体错误
- 不能生成md文件
- Excel 文件命名格式：`inventory_{YYYY-MM-DD}.xlsx`
